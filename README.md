VfxGraphAssets
==============

Basic asset collection for Unity VFX Graph

System requirements
-------------------

- Unity 2020.2 or later
- HDRP 10.0 or later

## How to install

[Follow those instructions] to set up the scoped registry. Then, you can install
the VFX Graph Assets package via Package Manager.
